WHAT NOT TO DO
================================================================================

❌ Don't use tableRef.current.getTableData() for modal items
   - Returns current page only, not all items
   - Will break bulk operations on paginated tables

❌ Don't close modal before showing success/error message
   - User won't see the feedback
   - They'll wonder if operation worked

❌ Don't skip error handling on API calls
   - Reset state on error: setAllCustomerItems([])
   - Show alert to user on failure

❌ Don't use page_size=1000 or page_size=10000 without pagination
   - Will lose data if more items exist
   - Should use while loop with has_next check
